## ğŸ® How to Play

### Objective
Slide tiles to combine matching numbers. Keep merging until you create the 2048 tile!

### Game Rules
- Swipe to move all tiles in one direction
- When two tiles with the same number touch, they merge into one
- After each move, a new tile (2 or 4) appears randomly
- Game ends when no more moves are possible
- Reach 2048 to win, but you can keep playing for higher scores!

### Scoring
- Each merge adds the new tile's value to your score
- Example: 2+2=4 adds 4 points
- Larger merges = more points
- Try to beat your high score!
